# User Interview Template

Present this checklist after scraping. Prefill found values with `[FOUND — please confirm]`.
Mark missing required fields with `[MISSING — required]` and optional with `[MISSING — optional]`.

---

```
=== WEBSITE GENERATOR: REVIEW INFORMATION ===

[ ] Company name: ___
[ ] Industry and category: ___
[ ] Main location (city + region/state): ___
[ ] Additional locations or service area (e.g. "entire metro area", "nationwide"): ___
[ ] Year founded: ___

SEO
[ ] Primary Keyword — REQUIRED (e.g. "dentist zurich", "plumber london"): ___
[ ] 2-4 Secondary Keywords (e.g. "emergency dentist", "teeth whitening"): ___
[ ] Main service for SEO focus: ___
[ ] Target audience (e.g. "families in north zurich", "small businesses"): ___
[ ] Competitor websites to check (optional): ___

[ ] Desired style — or "AI should decide": ___
[ ] Color scheme — or keep existing colors: ___
[ ] Logo available and should be kept? Yes/No: ___
[ ] Reference website the client likes (optional): ___

[ ] Which pages are needed? Standard: Home, About, Services, Contact: ___
[ ] Separate page per service? Yes/No: ___
[ ] Blog or news section? Yes/No: ___
[ ] Additional pages (Gallery, FAQ, Team, etc.): ___

[ ] Main website goal: Call / Form / Book appointment / Purchase: ___
[ ] Current promotion or offer to feature prominently? ___

[ ] Language(s): ___

=== WAIT FOR RESPONSE — DO NOT PROCEED TO PHASE 3 UNTIL ANSWERED ===
```

## After User Responds

Compile a PROJECT BRIEF and present it for final confirmation:

```
=== PROJECT BRIEF ===
Business: [name] — [industry]
Location: [city, region, country]
Primary Keyword: [keyword]
Secondary Keywords: [list]
Target Audience: [description]
Website Goal: [call / form / booking / purchase]
Pages planned: [list]
Language: [language(s)]
Design: [style or "AI decides"]
NAP: [name] | [address] | [phone]
=== END BRIEF ===
```

Ask: "Is this correct? Any changes before we proceed with keyword research?"
