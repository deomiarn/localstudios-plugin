# SEO Rules & Keyword Strategy

## Keyword-to-Page Assignment Rules

### Cluster Logic
Each page targets a semantic keyword cluster, not a single keyword. The principle:
one strong page beats five thin pages.

### Home Page Cluster
- Brand name + Primary keyword + City + Region
- "Near me" variations of primary service
- Trust keywords: "experienced", "certified", "since [year]"
- Geo variations: city, district, region all on one page

### Service Page Cluster (one per main service)
- [Service name] + [City]
- [Service name] + [District/Neighborhood]
- [Service name] + [Region/State]
- [Service name] + cost/price keywords (if applicable)
- [Service name] + related long-tail questions
- All geo-terms embedded naturally, NOT separate pages per location

### Contact Page Cluster
- [Service] + contact + [City]
- [Service] + opening hours
- [Service] + appointment
- [Service] + directions / how to reach
- [Service] + phone number

### About Page Cluster
- [Company name] + team
- [Company name] + experience / history
- [Service] + [City] + trusted / certified
- Local community keywords

## Geo-Cluster Embedding Rules

A page targeting "Dentist Zurich" should ALSO rank for:
- "Dentist Zurich North" → mention "Zurich North" naturally in content
- "Dentist Canton Zurich" → mention "Canton Zurich" in at least one H2 or paragraph
- "Dentist Oerlikon" → mention the district name in a local reference

Rules for natural embedding:
- Geo-term in the first paragraph (establishes local relevance)
- Geo-term in at least one H2 heading
- District/neighborhood names in "how to find us" or "serving" sections
- Region/state name in the business description
- Never repeat the same geo-term more than 3 times on a page

## Keyword Density Guidelines
- Primary keyword: 1-2% density (roughly 1x per 100 words)
- Secondary keywords: 0.5-1% each
- Geo-terms: mentioned 2-3 times naturally throughout
- NEVER keyword-stuff — if it reads awkwardly, rewrite

## Internal Linking Anchor Text Rules
- Use keyword-rich anchor texts: "[Service] in [City]" not "click here"
- Vary anchor texts — don't use the exact same text for every link to a page
- Every anchor text should read naturally in the sentence
- Acceptable patterns:
  - "Learn more about our [service]"
  - "[Service] in [City] — details"
  - "Our [service] team"
  - "Contact us for [service]"
